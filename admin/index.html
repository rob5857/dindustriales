Sincronización en la Nube (Firebase) No configurado

Cuando configuras Firebase, los pares Antes/Después se sincronizan automáticamente y todos los visitantes del sitio ven los cambios. Sin Firebase, cada navegador solo muestra lo guardado localmente.

  1. Crea un proyecto en Firebase Console.
  2. Activa Realtime Database (empieza en modo test o pega las reglas de abajo).
  3. En Authentication → método Correo/Contraseña → crea un usuario admin.
  4. Ve a Configuración del proyecto y copia el objeto firebaseConfig.
  5. Edita el archivo js/firebase-config.js con tus valores y el email del usuario admin.
  6. Sube los cambios y vuelve a iniciar sesión en el admin con la contraseña del usuario de Firebase.

Reglas recomendadas del Realtime Database:

{
  "rules": {
    ".read": true,
    "beforeafter": { ".write": "auth != null" },
    "gallery":     { ".write": "auth != null" }
  }
}
